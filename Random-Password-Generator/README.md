# Random Password Generator

## Description
This repository contains a Python-based Random Password Generator developed as part of the DecodeLabs Internship Program.
The application generates secure and random passwords based on the user-defined password length using Python's built-in random and string modules.
## Features
Generate Random Passwords
User-Defined Password Length
Uses Uppercase Letters
Uses Lowercase Letters
Uses Numbers
Input Validation
Generate Multiple Passwords
Exit Application
## Technologies Used
Python 3
Visual Studio Code
## Output
The application provides the following functionalities:
Accept password length from the user
Generate random passwords using letters and numbers
Display generated password instantly
Validate user input
Allow generation of multiple passwords
Exit the application safely
