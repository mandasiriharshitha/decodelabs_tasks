# Expense Tracker

## Description

This repository contains a Python-based Expense Tracker developed as part of the DecodeLabs Internship Program.

The application helps users record expenses, calculate total spending, and generate expense summaries through a simple menu-driven interface.

## Features

- Add Expenses
- View Expenses
- Calculate Total Spending
- Generate Expense Summary
- Exit Application

## Technologies Used

- Python 3
- Visual Studio Code

## Output

The application provides the following functionalities:

- Record expenses with details
- View all recorded expenses
- Calculate total spending
- Generate expense summaries
- Exit the application
